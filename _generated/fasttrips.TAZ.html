fasttrips.TAZ¶

class fasttrips.TAZ(input_dir, output_dir, today, stops, transfers, routes, is_child_process)[source]¶

TAZ class.

One instance represents all of the Transportation Analysis Zones as well as their access links and egress links.

Todo

This is really about the access and egress links; perhaps it should be renamed?

Stores access link information in TAZ.walk_access, and TAZ.drive_access, both instances of pandas.DataFrame.
